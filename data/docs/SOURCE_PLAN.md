# Recommended source plan

## Airports
Primary:
- OurAirports
- OpenFlights airports
- airport operator sites for premium summaries and terminal references

## Airlines
Primary:
- IATA / airline official websites
- OpenFlights airlines
- airline investor / fleet / cabin pages

## Routes
Primary:
- OpenFlights routes
- airline route maps
- airport destination maps

## Lounges
Primary:
- airline official lounge pages
- airport official lounge directories
- Priority Pass / LoungeKey for access patterns

## Aircraft
Primary:
- manufacturer pages
- airline fleet pages
- official seat / cabin pages where available

## Destinations
Primary:
- destination tourism boards
- airport / route connectivity context
- hotel / premium travel positioning

## Flight numbers
Primary:
- live provider or aviation API
- generated only when route and airline references are valid

## Important
Do not generate public SEO pages from random synthetic travel data. Use synthetic data only for testing the engine.
